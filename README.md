# Analyzing and Visualizing Regional Sales Performance

##  Project Overview

This project focuses on analyzing and visualizing regional sales performance using Microsoft Excel. The objective is to transform raw sales data into meaningful insights through data cleaning, analysis, and interactive dashboards.

The project helps identify sales trends, regional performance, profitability, and key business metrics to support data-driven decision-making.

---

##  Objectives

- Clean and prepare raw sales data for analysis
- Analyze sales performance across different regions
- Calculate important business KPIs
- Identify high-performing and low-performing regions
- Understand profit and sales trends
- Create an interactive Excel dashboard for visualization

---

##  Tools Used

- **Microsoft Excel**
   - Data Cleaning
   - Pivot Tables
   - Pivot Charts
   - Data Analysis
   - Dashboard Creation
